EditCSS for Firebird

Nov 21st, 2003 |

I just stubled across a neat enhancement for Firebird. Edit CSS allows you to edit the CSS of a web page on the fly. It’s great for when you want to see either how a web page did something or you want to change something in your web page. If I still did full time web development this would be my new best friend. You can find more information at http://editcss.mozdev.org/.
