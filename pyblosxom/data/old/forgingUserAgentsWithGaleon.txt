Forging user agents with Galeon

<!-- Date: 2002-11-24 05:04 -->
<p>
So, after the fiasco with <a href="http://www.ecollege.com/">ecollege.com</a> I decided to look up how to change my browsers user-agent (I use <a href="http://galeon.sf.net/">galeon</a>).  Here is how you do it (take from the Galeon Faq):

<ol>
<li>To get the current user agent type:
<pre>gconftool -g /apps/galeon/Advanced/Network/user_agent</pre></li>
<li>To set a user agent type:
<pre>gconftool -s /apps/galeon/Advanced/Network/user_agent --type=string "XXX"</pre>
For example, to make the system think that you're running IE 6.0 on WinXP do this:
<pre>gconftool -s /apps/galeon/Advanced/Network/user_agent --type=string "Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.1; .NET CLR 1.0.3705)"</pre></li>
<li>Finally, to restore the default type:
<pre>gconftool -s /apps/galeon/Advanced/Network/user_agent --type=string default</pre></li>
</ol>
</p>
